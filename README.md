Sales Forecasting App

The main aim of this project is to use a sales dataset from Kaggle to predict future sales. 
The model uses covariates such as yesterday_sales, last_week_sales, and week_sales_mean 
to predict future sales, which serve as the target variable. An XGBoost model is used 
to train and test the prediction system.

The application can answer questions such as:
- What is the average sales value for each category for a period of time?
- How many units should be stocked for a given time period based on past sales?
- How do sales trends evolve?

Users can also explore visual charts that display historical sales patterns and forecasted sales.
